# resuMade

A modern resume builder application with a clean, professional interface and dark/light theme support.

## Features

- **Personal Information** - Add your name, email, and phone number
- **Education** - Track multiple education entries with degree, institution, major, and graduation date
- **Work Experience** - Document your professional history with job titles, companies, years, and descriptions
- **Theme Toggle** - Switch between light and dark modes
- **Live Preview** - See your resume update in real-time as you edit
- **Edit & Delete** - Modify or remove entries with intuitive controls

## Tech Stack

- React 19.1.1
- Vite 7.1.0
- Tailwind CSS 4.1.12
- Radix UI components
- Lucide React icons

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint
